# Ali Shahbaz Portfolio

A modern and interactive personal portfolio website built with **React.js**, **Vite**, and **Tailwind CSS v4**. This project showcases my professional experience, technical skills, projects, and resume as a Full-Stack JavaScript Developer.

---

## 👨‍💻 About Me

**Ali Shahbaz** is a Full-Stack JavaScript Developer specializing in **React.js**, **Next.js**, and **Node.js**, with experience building scalable and high-performance web applications.

I focus on designing clean, maintainable front-end architectures and developing robust backend systems using modern JavaScript technologies. My expertise includes creating reusable components, optimizing application performance, and delivering seamless user experiences.

Passionate about modern web development, API-driven architectures, and scalable software solutions, I enjoy transforming complex requirements into production-ready applications through both educational and real-world projects.

## 🚀 Features

- Modern and responsive UI/UX

- Professional resume showcase

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio-ali.git
```

Navigate to the project directory:

```bash
cd portfolio-ali
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🔨 Available Scripts

### Development

```bash
npm run dev
```

Starts the Vite development server.

### Build

```bash
npm run build
```

Builds the application for production.

### Preview

```bash
npm run preview
```

Previews the production build locally.

### Lint

```bash
npm run lint
```

Runs ESLint to analyze and maintain code quality.
